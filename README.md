Welcome! 

Ce cours introduit les fondements du machine learning appliqué aux problématiques actuarielles. Il analyse l'évolution du métier d'actuaire face à la transformation numérique du secteur de l'assurance, expose les limites des approches statistiques classiques, et présente la taxonomie des méthodes d'apprentissage automatique. Des exemples concrets tirés de la pratique actuarielle illustrent chaque concept.
